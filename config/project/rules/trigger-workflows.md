# トリガーワード → ワークフロー

以下のトリガーワードを検知したら、**`WORKFLOWS.md`** の該当セクションを読んで処理する：

| トリガー                                 | ワークフロー   | WORKFLOWS.mdセクション     |
| ---------------------------------------- | -------------- | -------------------------- |
| 「自動化したい」「毎回これやるの面倒」等 | 自動化提案     | 自動化提案ワークフロー     |
| 「〇〇とご飯」「店を探して」等           | レストラン予約 | レストラン予約アシスタント |
| 返信案を作成する時                       | 返信案作成     | 返信案作成ワークフロー     |
| 日程調整の返信を作成する時               | 日程調整返信   | `/schedule-reply` コマンド |
| カレンダー登録する時                     | カレンダー登録 | カレンダー登録ワークフロー |
