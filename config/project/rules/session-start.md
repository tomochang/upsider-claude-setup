# セッション開始時に必ず読むファイル

セッション開始時（最初のメッセージを処理する前）に、以下を**実際に読む**こと。「参照」ではなく、Readツールで開いて中身を確認する：

1. **`SHARED_RULES.md`** — 予定・関係性の自動更新ルール、Todo振り分けルール等
2. **`private/INDEX.md`** — private/配下のファイル構成と連動ルール
3. **`SOUL.md`** — 振る舞いのルール（敬語、トーン、対外的な人格設定）
4. **`IDENTITY.md`** — ユーザーの代理として外部に返信する際の人格設定

**読んだ後、ログに記録:**

```bash
echo "$(date +%Y-%m-%d_%H:%M:%S) SESSION_START: files loaded" >> ~/workspace/logs/session_reads.log
```

## hookからのリマインダーを受け取ったら

`[MANDATORY_SESSION_PROTOCOL]` というリマインダーがコンテキストに含まれていたら、**ユーザーへの応答より先に**上記ファイルを読むこと。「遊ぼう」「こんにちは」等の軽い会話でも例外なし。ファイルを読んでログを記録してから応答する。
